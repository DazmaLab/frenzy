Настройка сети

Для настройки сети предназначена утилита “netconf“.

При вызове команды “netconf” без параметров появляется меню с выбором модуля (lan, ppp, vpn, adsl).

netconf lan

Модуль настройки локальной сети. После выбора сетевого интерфейса для настройки Вам будет предложено ввести для него IP-адрес, маску подсети, шлюз по умолчанию, DNS-серверы и имя домена.

Если какие-то из настроек уже были указаны, они будут подставлены в поля ввода по умолчанию.

Если в Вашей сети используется DHCP, то в поле ввода IP-адреса следует ввести “DHCP” или не вводить вообще ничего.

После ввода настроек они будут сохранены и автоматически применены.

  • В Frenzy 0.3 аналогичная утилита называлась “lan-config”.

netconf ppp

Модуль настройки dialup-соединения. При создании нового соединения нужно указать имя соединения, порт модема, тип авторизации (login или pap/chap), логин и пароль для подключения и телефоны дозвона (можно ввести несколько номеров по очереди, пустая строка - конец ввода).

После сохранения настроек в меню модуля появится пункт “Соединение с <имя соединения>”, при выборе которого начнется дозвон.

  • В Frenzy 0.3 аналогичная утилита называлась “ppp-config”.

netconf vpn

Модуль настройки vpn-соединения. При создании нового соединения нужно указать имя соединения, адрес сервера VPN, логин и пароль для подключения и ряд дополнительных настроек (использовать ли сжатие, шифрование, делать ли шлюзом по умолчанию).

После сохранения настроек в меню модуля появится пункт “Соединение с <имя соединения>”, при выборе которого начнется установка VPN-соединения.

Перед настройкой VPN-соединения нужно настроить локальную сеть.

netconf adsl

Модуль настройки adsl-соединения. При создании нового соединения нужно указать имя соединения, сетевой интерфейс, к которому подключен ADSL-модем, логин и пароль для подключения и ряд дополнительных настроек (делать ли шлюзом по умолчанию, включать ли NAT и DNS).

После сохранения настроек в меню модуля появится пункт “Соединение с <имя соединения>”, при выборе которого начнется установка ADSL-соединения.

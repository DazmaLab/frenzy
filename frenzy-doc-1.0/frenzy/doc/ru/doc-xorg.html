X Window system

Графическая оболочка может быть запущена автоматически при старте системы, если указать опцию gui в меню загрузчика. В уже загруженной системе запустить графическую оболочку можно командой startx.

В качестве графической оболочки используется fluxbox. Щелчок правой кнопкой мыши по рабочему столу вызывает меню приложений.

Переключение раскладки клавиатуры производится комбинацией клавиш Ctrl+Shift. Для переключения доступны два языка - английский и национальный язык, выбранный при загрузке (если вы выбрали lang=en, то язык будет только один). Если включен язык, отличный от английского, загорается индикатор ScrollLock.

Чтобы переключиться в текстовый режим, нажмите Ctrl+Alt+Fn (F1-F8). Чтобы вернуться в графическую оболочку, нажмите Alt+F9. Для принудительного завершения графической оболочки нажмите Ctrl+Alt+Backspace.

Горячие клавиши

Комбинация клавиш Действие
Alt+F1, Alt+F2, Alt+F3, Alt+F4 Переключение рабочих столов
Winkey, Ctrl+Esc Вызов меню приложений
Alt+Esc Вызов меню рабочего стола
Ctrl+Стрелки доп. клавиатуры Переместить окно
Ctrl+Alt+Стрелки доп. клавиатуры Изменить размер окна
Ctrl+Alt+R Утилита запуска приложений
Ctrl+Alt+O Запустить Opera
Ctrl+Alt+L Запустить Sylpheed
Ctrl+Alt+Q Запустить GQview
Ctrl+Alt+T Запустить терминал rxvt
Ctrl+Alt+S Снять скриншот (картинка будет сохранена в /root/MyFiles)
Ctrl+Alt+Esc Вызвать xkill для завершения программы

Проблемы при запуске

В случае возникновения проблем при запуске графической оболочки с настройками по умолчанию попробуйте удалить /etc/X11/xorg.conf и запустите утилиту xautoconf самостоятельно:

  • xautoconf safe - использует в качестве видеодрайвера стандартный VESA-драйвер.
  • xautoconf safemon - использует найденный видеодрайвер, но настройки монитора берет стандартные. Это полезно, когда монитор возвращает неверные данные о частоте развертки.

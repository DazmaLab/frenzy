Frenzy 1.0 (Dreamchild)
------------------------------------------

Frenzy - это "портативный инструмент системного администратора" на 
базе ОС FreeBSD, который удобно постоянно иметь при себе. Система 
содержит набор программного обеспечения для настройки, проверки и 
анализа сети, тестирования компьютерного "железа" и ряда других задач.

*** Лицензии

Frenzy распространяется по BSD-лицензии, поскольку
основана на FreeBSD.

Различные программы, входящие в состав Frenzy, 
распространяются в соответствии с лицензиями на них.
